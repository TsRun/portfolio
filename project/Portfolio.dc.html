Foreman

a personal task queue for AI coding agents une file de tâches personnelle pour agents de code IA
Tauri 2 · Rust · React 18 · SQLite private repo — ask me for a demo dépôt privé — demandez une démo

A macOS desktop app that turns plain-English intent into shipped code: queue tasks under a project, let a classifier route them to specialised agents, and watch them run — in parallel across agents, strictly serial within one. Une app de bureau macOS qui transforme une intention en code livré : des tâches en file sous un projet, un classifieur qui les route vers des agents spécialisés, et une exécution en parallèle entre agents — strictement séquentielle au sein d'un même agent.

Type a goal, get a planUn objectif tapé, un plan

Every task is auto-classified by Claude Haiku or Gemini Flash: a complexity grade plus a route to the right role agent — Frontend, Backend, Tests. Low-complexity work runs itself; bigger jobs wait for a click. Chaque tâche est classée par Claude Haiku ou Gemini Flash : un niveau de complexité et un agent de rôle — Frontend, Backend, Tests. Le simple s'exécute tout seul ; le reste attend un clic.

02

A provider per agentUn provider par agent

Claude, Codex, Gemini or Aider — each agent runs on the CLI you choose, spawned in the project directory, its output streamed back as a conversation with token costs tracked per run. Claude, Codex, Gemini ou Aider — chaque agent tourne sur le CLI de votre choix, lancé dans le dossier du projet, sa sortie streamée comme une conversation, coût en tokens suivi par run.

03

The roomLa room

Agents coordinate: hand a sub-task to a teammate, depth-bounded so delegation never spirals. Parallel across agents, serial within one — a session can't do two things at once. Les agents se coordonnent : une sous-tâche se délègue à un coéquipier, avec un garde-fou de profondeur. Parallèle entre agents, séquentiel au sein d'un agent — une session ne fait qu'une chose à la fois.

Un backend Rust sous Tauri 2, un front React 18 en CSS simple.
